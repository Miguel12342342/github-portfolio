Sobre Mim

Sou Miguel Angelo, Desenvolvedor Flutter Mobile (Junior/Pleno), com 3 anos de experiência especializada na construção de soluções móveis nativas de alta performance para Android e iOS. Minha paixão é transformar requisitos complexos em interfaces de usuário intuitivas e códigos escaláveis. Eu gerencio o ciclo de vida completo de um app, do conceito e design de experiência (UX/UI) até a implantação e manutenção na Play Store e App Store.

Minha expertise e foco em projetos incluem:

  • API REST & Dados: Implementação robusta de APIs REST, garantindo comunicação eficiente e manipulação segura de dados JSON em larga escala.
  • Gerenciamento de Estado: Proficiência em arquiteturas como BLoC e Provider, resultando em código modular, fácil de testar e sustentável a longo prazo.
  • Backend As a Service (BaaS): Experiência prática com Firebase (Firestore e Authentication), agilizando o desenvolvimento de backends e funcionalidades em tempo real.
  • Versionamento: Sólidas práticas de desenvolvimento colaborativo utilizando Git e GitHub (Branching, Merge Requests e Code Reviews).

Busco ativamente novos desafios e prospero em ambientes colaborativos que incentivam a inovação. Meu foco é entregar produtos digitais que resolvam problemas reais e gerem valor mensurável para os usuários e o negócio.
